ğŸŒ Live UI Preview

The Gradio interface provides a sketchpad where you can draw digits and get real-time predictions with confidence scores.

How it works:

  1. Draw a digit (0-9) in the sketchpad
  2. The model processes your input (resized to 28x28, inverted, normalized)
  3. Predictions show top 3 probable digits with confidence percentages
