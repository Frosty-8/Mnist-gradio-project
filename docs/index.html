🔢 MNIST Digit Recognition

An interactive web application built with TensorFlow and Gradio that recognizes handwritten digits using a convolutional neural network (CNN).

📌 Project Overview

This application uses a trained CNN model to predict handwritten digits (0-9) with high accuracy. Key aspects include:

  • Built with TensorFlow 2.19 and Keras
  • Uses the classic MNIST dataset (60,000 training images)
  • CNN architecture with 2 convolutional layers and dropout
  • Gradio interface for easy interaction
  • Model achieves ~99% validation accuracy

🧮 Model Architecture

The neural network consists of the following layers:

  • Conv2D(32, (3,3), activation='relu'
  • MaxPooling2D((2,2))
  • Conv2D(64, (3,3), activation='relu'
  • MaxPooling2D((2,2))
  • Flatten()
  • Dense(128, activation='relu')
  • Dropout(0.5)
  • Dense(10, activation='softmax')

🌐 Live UI Preview

The Gradio interface provides a sketchpad where you can draw digits and get real-time predictions with confidence scores.

How it works:

  1. Draw a digit (0-9) in the sketchpad
  2. The model processes your input (resized to 28x28, inverted, normalized)
  3. Predictions show top 3 probable digits with confidence percentages

🚀 How to Run

  1. Clone the project: git clone https://github.com/Frosty-8/frosty-8-mnist-gradio-project.git
  2. Install dependencies: pip install -r requirements.txt or pip install gradio tensorflow numpy
  3. Run the application: python main.py
  4. Open in browser: http://localhost:8080

Requirements:

  • Python ≥ 3.12
  • TensorFlow 2.19.0
  • Gradio ≥ 5.34.2
  • NumPy ≥ 2.1.0
